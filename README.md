# Employee Attrition Prediction using Machine Learning

## Overview

This project predicts employee attrition using supervised machine learning techniques to help organizations identify employees who are at risk of leaving. The objective is to analyze workforce data, discover the key factors influencing attrition, and build predictive models that support data-driven HR decision-making.

---

## Problem Statement

Employee attrition leads to increased recruitment costs, productivity loss, and knowledge gaps within organizations. This project develops a machine learning solution that predicts employee attrition based on employee demographics, job-related attributes, and workplace factors, enabling proactive retention strategies.

---

## Objectives

- Perform data preprocessing and exploratory data analysis (EDA).
- Identify key factors contributing to employee attrition.
- Train and compare multiple machine learning models.
- Evaluate model performance using standard classification metrics.
- Visualize insights through Power BI dashboards and analytical charts.

---

## Key Findings

- Employees with lower monthly income and shorter tenure showed a higher probability of attrition.
- Work-life balance and job satisfaction significantly influenced employee retention.
- Logistic Regression achieved the best overall performance with a ROC-AUC score of approximately **79.9%**.
